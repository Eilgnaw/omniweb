# 锁屏卡片

锁屏卡片和桌面卡片共用同一套编辑器、组件、占位符和 JS API。区别在于:锁屏不能在系统侧原地长按编辑,需要先把卡片位添加到锁屏,再回到 Omni 选择绑定哪张锁屏卡片。

## 支持尺寸

| 类型 | `dimension` | 画布尺寸 | 适合内容 |
|---|---|---|---|
| 锁屏 1×1 | `lock_1*1` | 76×76 vp | 图标、数字、短状态 |
| 锁屏 1×2 | `lock_1*2` | 160×76 vp | 图标 + 两行文字、小指标 |

新建时打开 Omni → 首页右上角 `+` → 类型选 **锁屏** → 选择尺寸 → 创建。

## 添加到锁屏

1. 在系统锁屏界面双指捏合进入编辑态
2. 点一个卡片位,添加本应用的锁屏卡片
3. 首次添加后,锁屏上会显示「点击选择卡片」占位
4. 点这个占位,Omni 会打开「选择锁屏卡片」页面
5. 选择一张同尺寸的锁屏卡片,它会立即应用到这个锁屏位

如果点锁屏占位没有拉起 App,可以从 Omni 内进入 **设置 → 锁屏卡片**,点已添加的锁屏位手动选择或更换卡片。

## 没有同尺寸卡片时

「选择锁屏卡片」页面只列出和当前锁屏位同尺寸的卡片。如果列表为空,点 **新建并使用** 会创建一张同尺寸锁屏卡片、绑定到当前锁屏位,并进入编辑器继续设计。

## 从卡片应用到锁屏位

如果你已经做好一张锁屏卡片,也可以走卡片优先的流程:

1. 首页顶部切到 **锁屏**
2. 长按要使用的卡片
3. 点 **应用到锁屏**
4. 选择一个同尺寸、已经添加到系统锁屏上的卡片位

这条路径适合一张卡要放到多个锁屏位,或临时替换某个锁屏位当前绑定的卡片。

## 注意事项

- 锁屏位和卡片必须同尺寸,`lock_1*1` 只能选 `lock_1*1`,`lock_1*2` 只能选 `lock_1*2`
- 首次添加时看到的「点击选择卡片」只是占位入口,不会出现在卡片列表里
- 已选择的锁屏卡片会在刷新、重启或应用更新后保持绑定;如果绑定的卡片被删除,会回到占位状态
- 锁屏卡片不适合高频刷新,需要稳定周期刷新时优先做桌面卡片
