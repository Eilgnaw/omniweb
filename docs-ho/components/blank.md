# 占位

占位用来撑开剩余空间。横排里撑横向、纵列里撑纵向,常用于把后续组件推到另一端。

## 用法

最常见的场景:把同一行里的两个内容推开:

```text
横排
 ├─ 文本,「内容」填 标题
 ├─ 占位
 └─ 文本,「内容」填 26°
```

显示:

```text
┌─────────────────────┐
│ 标题            26° │
└─────────────────────┘
```

## 属性

| 字段 | 可填写 | 说明 |
|------|------|------|
| 最小占位 | `10` / `10px` / `10vp` | 最小占位大小;不支持百分比 |
| 显示 | `true` / `false` / `${show}` | 控制是否显示 |

:::warning 不支持百分比 / 负值
「最小占位」写 `50%` 或负数会被忽略。占位的语义是“至少多大,实际可以更大”,不适合用百分比表达。
:::

## 例:三列均分

```text
横排
 ├─ 文本,「内容」填 A
 ├─ 占位
 ├─ 文本,「内容」填 B
 ├─ 占位
 └─ 文本,「内容」填 C
```

两个占位会均分剩余空间,A / B / C 三等分对齐。

## 注意事项

:::tip 做规整布局优先用占位
占位没有宽、高、内边距、背景色等通用字段。做规整布局时,优先用占位推开内容,不要手算每个子组件的宽度百分比。
:::
