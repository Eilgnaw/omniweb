# 间隔 Spacer

弹性占位,把同容器里前后的元素**推开**。理解为"会自动扩展的空白"。

## 用法

从编辑器右上角 ➕ 添加 **spacer** 即可,通常不需要任何属性。

## 例:左侧文字 + 右侧图标

在 **hstack** 里放:

```
[ text ] [ spacer ] [ icon ]
```

效果:文字贴左、图标贴右、中间被 spacer 撑开。

## 例:垂直布局把内容顶满

在 **vstack** 里放:

```
[ text 标题 ]
[ spacer ]
[ text 副标题 ]
```

效果:标题贴顶、副标题贴底。

:::tip
- 没有 spacer,容器内的子元素默认**居中**或按"对齐方式"排列。
- 同一个容器里可以放多个 spacer,它们会**平分**剩余空间。
:::
