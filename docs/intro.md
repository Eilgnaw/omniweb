# 欢迎使用 Omni Widgets

Omni Widgets 是一款让你**自己 DIY iOS 小组件**的工具 App。无需写代码,你也能拼出一个漂亮的小组件;
想做更复杂的、能联网拉数据的、能跟你互动的小组件? Omni 内置了一个完整的 JavaScript 运行时,你可以在小组件里写脚本。

## Omni 能做什么

- **9 种小组件类型**:桌面小、中、大;锁屏圆形、矩形、内联;控制中心按钮、开关;实时活动(灵动岛)。
- **可视化编辑器**:文本、图片、SF Symbol、按钮、形状、容器…… ➕ 一下就能加上去。
- **JavaScript 接口**:可以联网请求、读取天气、定位、健康、日历、文件,还能动态控制小组件刷新和实时活动。
- **占位符语法**:用 `{变量}` 把 JS 拿到的数据,直接绑定到组件上显示。
- **iCloud 同步、URL Scheme、快捷指令**:跟系统深度配合。

## 遇到问题

- 小组件**不刷新**? 这是 iOS 系统行为,详见 [小组件刷新控制](./api/control.md)。
- JS **报错**? 在编辑器里打开"控制台"查看 `console.log` 输出。
